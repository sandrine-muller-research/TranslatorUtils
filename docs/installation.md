# Installation

## Stable release

To install TranslatorUtils, run this command in your terminal:

```sh
uv add TranslatorUtils
```

Or if you prefer to use `pip`:

```sh
pip install TranslatorUtils
```

## From source

The source files for TranslatorUtils can be downloaded from the [Github repo](https://github.com/sandrine-muller-research/translatorutils).

You can either clone the public repository:

```sh
git clone git://github.com/sandrine-muller-research/translatorutils
```

Or download the [tarball](https://github.com/sandrine-muller-research/translatorutils/tarball/master):

```sh
curl -OJL https://github.com/sandrine-muller-research/translatorutils/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd translatorutils
uv pip install .
```
